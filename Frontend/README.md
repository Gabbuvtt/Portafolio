# Gabriel Mejia — Portafolio

Sitio personal de **Gabriel Mejia**: Arquitecto de Software, Desarrollador Fullstack y Analista de Datos/IA. Co-fundador de **BiCloud M** y **Bolix**.

## Stack
- **Frontend:** React 18, TypeScript, Vite, TailwindCSS, shadcn/ui
- **Tipografías:** Space Grotesk, JetBrains Mono

## Desarrollo

```bash
npm install
npm run dev      # http://localhost:5000
npm run build    # genera /dist
```
